Lecture 01-Perceptron

Forward Propagation

  • How neural networks function?
    1. Data in.
    2. Doing dot product with weights.
      • Each input has it's own weight for a single function.
    3. Summing the result.
    4. Plugging into a "non-Linear" activation function.
    5. Adding Bias.
      • Each Activation function has its own bias.
    6. Result out.

Activation Function

  • Why is non-linear activation function needed?
    • Nonlinearity allows neural networks to develop complex representations and functions based on the inputs that would not be possible with a simple linear regression model.
  • Common Activation Functions
    • Sigmoid Function
    • Hyperbolic Tangent
    • Rectified Linear Unit(ReLU)

Multi-Output Perceptron

  • Dense Layer(Fully Connected Layer)
    • How many "" is needed for a fully connected layer with "" inputs and "" activation functions?

Double-layer Neural Network

  • Activation function is "definitely" needed! Due to the principle of "superposition", multiple layers with nonlinearity will be considered as one layer.

Deep Neural Network

  • A network with multiple layers is called "deep neural network".

Quantifying Loss

  • A "loss function" is needed to know how well-trained the model is.

  • Empirical Loss

    • It messures the total loss over our entire dataset.

  • Binary Cross Entropy Loss

    • Can be used with models that output a probability between 0 and 1.

  • Mean Squared Error Loss

    • Can be used with regression models that output continuous real numbers.

Loss Optimization

  • We want to find the network weights that achieve the lowest loss.
  • Gradient Descent
    • To find the lowest loss.
    1. Initialize weights randomly.
    2. Loop until convergence.
    3. Compute gradient
    4. Update weights, <- (=>Learning rate )
    5. Return weights.
