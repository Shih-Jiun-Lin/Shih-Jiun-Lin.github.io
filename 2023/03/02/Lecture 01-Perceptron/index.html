Lecture 01-Perceptron

Forward Propagation

  • How neural networks function?
    1. Data in.
    2. Doing dot product with weights.
      • Each input has it's own weight for a single function.
    3. Summing the result.
    4. Plugging into a "non-Linear" activation function.
    5. Adding Bias.
      • Each Activation function has its own bias.
    6. Result out.

Activation Function

  • Why is non-linear activation function needed?
    • Nonlinearity allows neural networks to develop complex representations and functions based on the inputs that would not be possible with a simple linear regression model.
  • Common Activation Functions
    • Sigmoid Function
    • Hyperbolic Tangent
    • Rectified Linear Unit(ReLU)
